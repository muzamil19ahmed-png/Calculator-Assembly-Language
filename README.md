> A simple calculator implemented in x86 Assembly Language using MASM and Irvine32 library.

---

## 📖 Overview

This project implements a basic calculator in **x86 Assembly Language** as part of the Computer Organization and Assembly Language course at Bahria University.

### Supported Operations

| Operation | Description |
|-----------|-------------|
| Addition | Adds two numbers |
| Subtraction | Subtracts second number from first |
| Multiplication | Multiplies two numbers |
| Division | Divides first number by second |

---

## ✨ Features

- User-friendly menu interface
- Input validation for invalid operations
- Division by zero protection
- Modular design with separate procedures

---

## 🛠️ Technology Stack

| Technology | Purpose |
|------------|---------|
| **MASM** | Microsoft Macro Assembler |
| **Irvine32** | Input/Output Library |
| **x86 Assembly** | Low-level programming |
